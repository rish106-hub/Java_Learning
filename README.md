## 🚀 Getting Started

### Requirements
- Java JDK (version 17+ recommended)  
- IntelliJ IDEA (or any other Java IDE)

### Run a file
1. Clone the repo or download ZIP  
2. Open in IntelliJ IDEA  
3. Navigate to `src/Main.java`  
4. Click the green ▶️ to run
